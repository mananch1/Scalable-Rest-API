
# MERN Stack Internship Assignment

## Overview
This repository contains a full-stack MERN application developed for the Backend Developer Intern assignment. It features a scalable REST API with authentication, role-based access control, and a responsive frontend dashboard.

## Features
- **Authentication**: User registration and login with JWT and password hashing.
- **Role-Based Access**: Granular permissions for Users and Admins.
- **Task Management**: CRUD operations for tasks (Task entity).
- **Security**: Secure password handling, Input validation.
- **Documentation**: Swagger UI for API visualization.

## Tech Stack
- **Backend**: Node.js, Express, Mongoose, MongoDB.
- **Frontend**: React (Vite), Vanilla CSS.
- **Tools**: Swagger, Postman.

## Deployment
- **Frontend**: Vercel / Netlify
- **Backend**: Railway / Render / AWS EC2 (Scalable Node.js Server)

## Setup Instructions
1. **Backend**: Navigate to `server/`, run `npm install`, then `npm run dev`.
2. **Frontend**: Navigate to `client/`, run `npm install`, then `npm run dev`.
